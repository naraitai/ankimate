About

+

Anki is an incredibly useful tool for acquiring a range of knowledge, including foreign languages. The problem is it takes a lot of work to create good quality and personalised study cards.

Ankimate is intended to help reduce this workload for students of Japanese and Mandarin Chinese.

It's operation is simple. Upload a file containing vocabulary you intend to learn. Set some basic settings. Review the returned data. Re-search the database if desired. Download data and open it in Anki

Ankimate is under active development and features will be added over time. You can review and track releases below or suggest improvements here.

+
Current
  • Upload vocabulary file.
  • Set language (Japanese or Mandarin Chinese).
  • Toggle English translation
  • Set front and back values for basic Anki card creation.
  • Set value for one extra Anki card field
  • Reload sentences (if available)
  • Download file for import into Anki

This project makes use of data and resources created by others. What follows is a list of those resources as well as links to these resouces.

Tatoeba:

A searchable database of sentences and translations in a number of languages. CC-BY(2.0)

JMDict:

A Japanese dictionary project. CC BY-SA(4.0)

Leeds Frequency Corpora:

A frequency list based on internet corpora. CC BY(2.5)

Fugashi:

A Japanese tokenizer for Python to aid natural language processing (NLP). Usage guide

Jieba:

A Mandarin Chinese tokenizer for Python to aid natural language processing (NLP). Project information
